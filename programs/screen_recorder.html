How to Record Your Screen?

  1. Click the large "Start Recording" button.
  2. A browser popup will ask you what to share:
    • Entire Screen: Best for tutorials showing multiple apps.
    • Window: Record just one application (e.g., PowerPoint).
    • Browser Tab: Best for recording YouTube or websites.
  3. Enable Audio: Make sure to check the "Share system audio" box if you want to record computer sounds.
  4. Click "Share" to start recording.
  5. When finished, click "Stop". You can preview your video and download it as a WebM file.

Troubleshooting & Tips

Why is there no sound?

To record system audio (YouTube, Music, Games), you must select "Entire Screen" or "Browser Tab" and check the "Share system audio" box in the popup. MacOS has stricter permissions for system audio.

Which format is the video?

Videos are saved in .WebM format by default, which is optimized for web browsers. You can play it in Chrome, VLC, or convert it to MP4 later using a video converter.

Does it work on Mobile?

Screen recording APIs are currently limited to Desktop browsers (Chrome, Edge, Firefox, Opera). Mobile browsers (Android/iOS) generally block web-based screen recording for security reasons.
